SVG Editor - available functions & variables

  • No self-closing tags
  • framenr - 0+
  • steps - total frames
  • v - viewBox width
  • h - viewBox height
  • pulse( start , mid , [end=start] ) - pulse from start to mid in one cycle - returns value
  • scale( start , mid , [end=start] ) - scale (from center) - returns matrix string
  • ease( length , [framecount=steps/2] ) - ease from 0 to length in [framecount] cycle, and back to 0 again - returns value
  • minmax( min , max ) - use value between min and max - returns value
  • arcpath( [x1,y1 ] , [x2,y2] , [c1x,c1y , c2x,c2y ] ) - arc between 2 points with c-control points, c2x,c2y == c1x,c1y when not specified value
  • round( value , digits=1 ) - round, for smaller output
